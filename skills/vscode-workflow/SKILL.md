# VS Code Workflow

VS Code is the primary human control surface.

Preferred behavior:
- Keep files organized for easy navigation.
- Recommend `code .`.
- Prefer Makefile targets and scripts for terminal tasks.
- Keep README and docs current.
- Use `.vscode/tasks.json` only when it helps and stays non-invasive.

Recommended tasks:
- validate config
- run doctor
- show git diff
- backup live config
